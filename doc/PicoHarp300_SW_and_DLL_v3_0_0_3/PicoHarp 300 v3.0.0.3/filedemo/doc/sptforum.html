SPT Forum

For support as well as especially to encourage networking between SPT users, PicoQuant runs a forum for SPT-related discussions:   http://forum.picoquant.com/.

Aim and Purpose

The SPT forum is intended as a platform for users of PicoQuant's SymPhoTime 64 software package. It is not strictly limited to software related questions. As SymPhoTime 64 covers a wide range of applications from single molecule experiments to life sciences, discussions of the scientific background are of course welcome. Please note that not all content is public. Some topics are only visible after you login/register.
